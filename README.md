# ERP Image Sequence Processor

Convert image rows into grouped ERP SKU image columns.

## Input Format

| File Name | Direct Image Link |
|---|---|
| 3775S101 (1).jpg | https://... |

## Output Format

| erp_sku | Image 1 | Image 2 |
|---|---|---|
| 3775S101 | link | link |

## Features

- Auto ERP SKU extraction
- Auto sequence sorting
- Missing sequence handling
- CSV export
- Fully browser-based
- GitHub Pages compatible

## Deployment

Upload project to GitHub.

Enable GitHub Pages:

Settings → Pages → Deploy from branch → main → root
